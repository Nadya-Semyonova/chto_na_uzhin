# React 19.2.0 + TypeScript 5.9.3 + Vite 7.3.1

После выполнения работы провести проверки:

Проверить все файлы на ошибки
npm run lint

Автоматически исправить ошибки (где возможно)
npm run lint:fix

Отформатировать все файлы в проекте
npm run format
