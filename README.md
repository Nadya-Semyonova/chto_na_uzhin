# "Что на ужин?"

Интерактивное веб-приложение, созданное для решения ежедневной проблемы выбора блюда.  
Помогает определиться с ужином, предлагая случайные рецепты с учётом доступных продуктов в холодильнике.

---

## Основная идея

- Простые рецепты с доступными ингредиентами  
- Генерация случайных блюд  
- Фильтрация по продуктам из холодильника  
- Адаптивный дизайн

## Функциональность

### Текущая реализация

- **Просмотр рецептов**  
  Отображение списка рецептов с детальной информацией: название, ингредиенты, сложность, время приготовления.

- **Фильтрация по ингредиентам**  
  Выбор продуктов из холодильника → показ только доступных рецептов.

- **Генерация случайного рецепта**  
  Кнопка для получения случайного блюда с учётом активных фильтров.

- **Хранение данных**  
  Локальное хранение в `localStorage` — выбранные ингредиенты сохраняются.

## Компоненты

### `Sidebar`
Боковая панель с фильтрами по ингредиентам.  
**Пропсы:** `ingredients`, `selectedIngredients`, `onIngredientToggle`  
**Состояние:** управление выбором ингредиентов

### `RecipeCard`
Карточка с информацией о рецепте.  
**Пропсы:** `recipe`, `onSelect`  
**Отображает:** название, ингредиенты, сложность, время приготовления

### `GenerateButton`
Кнопка для генерации случайного рецепта.  
**Пропсы:** `onGenerate`, `disabled`  
**Состояние:** загрузка, активность

### `Layout`
Основной макет приложения.  
**Пропсы:** `children`  
**Отвечает за:** расположение боковой панели и контента

### `BottomSheet`
Выезжающая панель снизу для мобильной версии.  
**Пропсы:** `isOpen`, `onClose`, `children`  
**Состояние:** открыт/закрыт, позиция перетаскивания, анимация закрытия  
**Отвечает за:** отображение холодильника на мобильных устройствах, обработку свайпов, блокировку скролла фона

### `BurgerMenu`
Бургер-меню для мобильной версии.  
**Пропсы:** `isOpen`, `onClick`  
**Состояние:** открыто/закрыто, анимация иконки  
**Отвечает за:** переключение видимости `BottomSheet`, анимацию превращения в крестик, отображение только на экранах ≤768px
