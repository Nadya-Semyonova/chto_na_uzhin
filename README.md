# "Что на ужин?"
интерактивное веб-приложение, созданное для решения ежедневной проблемы выбора блюда. Приложение помогает пользователям определиться с ужином, предлагая случайные рецепты с учетом доступных продуктов в холодильнике.

# Основная идея
-Простые рецепты с доступными ингредиентами
-Генерация случайных блюд
-Фильтрация по продуктам из холодильника
-Адаптивный дизайн (в разработке)

# Функциональность
Текущая реализация: 
-Просмотр рецептов 
-Отображение списка рецептов
-Детальная информация: название, ингредиенты, сложность, время приготовления
-Фильтрация по ингредиентам: 
  -Выбор продуктов из холодильника
  -Показ только рецептов, доступных из выбранных ингредиентов
-Генерация случайного рецепта:
  -Кнопка для получения случайного блюда
  -Учет фильтров при генерации
-Хранение данных:
  -Локальное хранение в LocalStorage
  -Сохранение выбранных ингредиентов


# Компоненты
# Sidebar
Боковая панель с фильтрами по ингредиентам.
Пропсы: ingredients, selectedIngredients, onIngredientToggle
Состояние: управление выбором ингредиентов

# RecipeCard
Карточка с информацией о рецепте.
Пропсы: recipe, onSelect
Отображает: название, ингредиенты, сложность, время приготовления

# GenerateButton
Кнопка для генерации случайного рецепта.
Пропсы: onGenerate, disabled
Состояние: загрузка, активность

# Layout
Основной макет приложения.
Пропсы: children
Отвечает за: расположение боковой панели и контента
